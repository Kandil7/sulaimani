## 🎨 Design System Foundation

### Typography Scale
| Level | Size | Weight | استخدام |
|-------|------|--------|---------|
| Display | 32px | 700 | اسم المحل في الـ Splash |
| H1 | 26px | 700 | عناوين الصفحات |
| H2 | 20px | 600 | عناوين الأقسام |
| H3 | 16px | 600 | عناوين الكروت |
| Body L | 15px | 400 | النصوص الأساسية |
| Body M | 13px | 400 | النصوص الثانوية |
| Label | 12px | 500 | الـ Tags والـ Badges |
| Caption | 11px | 400 | التواريخ والـ Hints |

**الفونت:** Cairo — بيشتغل ممتاز عربي وإنجليزي معاً

***

### Color SystemARR
| Token | Color | استخدام |
|-------|-------|---------|
| Primary | #2E7D32 | أزرار، Active states، أيقونات رئيسية |
| Primary Surface | #E8F5E9 | خلفيات الـ Badges الخضرا |
| Secondary | #1565C0 | أدوية، إجراءات ثانوية |
| Secondary Surface | #E3F2FD | خلفية Badge أدوية |
| Danger | #D32F2F | منتج منتهي، حذف |
| Danger Surface | #FFEBEE | خلفية تنبيه منتهي |
| Warning | #F57C00 | قريب الانتهاء، مخزون منخفض |
| Warning Surface | #FFF3E0 | خلفية تنبيه تحذيري |
| Sidebar | #1A2332 | خلفية الـ Sidebar |
| Background | #F5F7FA | خلفية الصفحات |
| Surface | #FFFFFF | خلفية الكروت |
| Border | #E0E6ED | حدود الكروت والـ Dividers |
| Text Primary | #1A2332 | النصوص الأساسية |
| Text Secondary | #6B7C93 | النصوص الثانوية |

***

### Spacing & Radius System
| Token | Value | استخدام |
|-------|-------|---------|
| xs | 4px | مسافات داخل الـ Badges |
| sm | 8px | مسافة بين العناصر المتقاربة |
| md | 16px | Padding داخلي للكروت |
| lg | 24px | Padding الصفحات |
| xl | 32px | مسافة بين الأقسام |
| radius-sm | 6px | Badges, Tags |
| radius-md | 8px | Input fields, Buttons |
| radius-lg | 12px | Cards |
| radius-xl | 16px | Dialogs |

***

### Elevation & Shadow
- **Level 0:** Border فقط (1px solid Border) — Cards عادية
- **Level 1:** `shadow: 0px 2px 8px rgba(0,0,0,0.06)` — Cards hover
- **Level 2:** `shadow: 0px 4px 16px rgba(0,0,0,0.10)` — Dialogs
- **Level 3:** `shadow: 0px 8px 32px rgba(0,0,0,0.16)` — Dropdowns

- **Skeleton Loading** بدل Spinner في الجداول والكروت

### Empty States


- تظهر أعلى يمين الشاشة

- ❌ خطأ: خلفية Danger Surface، 5 ثواني

- Auto dismiss مع Progress bar

| Hover | Primary + 10% darker |
